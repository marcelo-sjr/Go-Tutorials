## 🚀 Getting Started

### Prerequisites

- Go 1.24+ (or your installed version)

Verify your installation:

```bash
go version
```

Clone the repository:

```bash
git clone https://github.com/marcelo-sjr/Go-Tutorials.git

cd Go-Tutorials
```

Run any example:

```bash
cd hello
go run .
```

Or build it:

```bash
go build
```
